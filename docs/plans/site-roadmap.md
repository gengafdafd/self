# 站点建设路线

## 背景

当前仓库已经完成基础初始化，但内容结构还需要固定下来，避免后续页面、文档和资料堆积混乱。

## 目标

- 建立统一目录结构
- 建立首页导航入口
- 建立个人介绍与项目展示区
- 为后续文档和内容更新留出稳定扩展空间

## 分阶段路线

### 第一阶段

- 完成首页基础信息
- 配置自定义域名与 HTTPS
- 固化 `docs/` 目录规范
- 增加个人介绍和目录入口

### 第二阶段

- 增加项目页
- 增加写作页
- 增加联系页
- 为每个栏目建立标准模板

### 第三阶段

- 增加 SEO 信息
- 增加社交分享信息
- 增加访问统计
- 视内容增长情况考虑迁移到更强的站点框架

## 下一步

- 完善首页真实文案
- 补齐项目展示内容
- 确认正式自定义域名配置
